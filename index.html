Apkshellext2

Show .apk icons in windows explorer

Install Instruction Latest ApkShellext2.dll Open Source

This is a homebrew trying to help people who like to collect andriod apps in windows

[Features]
  • Show Icon of apk files on windows explorer
  • Show Package information in Infotip
  • Batch rename apk as app label + version
  • Go to app google play page
